# Product Listing App 🛍️

A modern, responsive product listing application built with React and Vite. This project fetches dynamic product data from a public API and displays it in an interactive, aesthetically pleasing grid.

## 🚀 Features

- **Real-time Data Fetching**: Retrieves product information from the FreeAPI random products endpoint.
- **Responsive Design**: Mobile-friendly layout that adjusts seamlessly to different screen sizes.
- **Rich Product Cards**: Displays detailed information including:
  - Product Title & Description
  - Pricing & Discount Percentages
  - Ratings & Brand Info
  - High-quality Images
- **Fast Performance**: Built with Vite for ultra-fast development and optimized production builds.

## 🛠️ Technologies Used

- **React 19**: Modern UI library for building interactive interfaces.
- **Vite**: Next-generation frontend tooling.
- **CSS3**: Custom styles with a focus on premium aesthetics and responsive layouts.
- **FreeAPI**: Source for random product data.

## 📦 Installation & Setup

1. **Clone the repository**:
   ```bash
   git clone https://github.com/Ayandi999/productLising.git
   ```

2. **Navigate to the project directory**:
   ```bash
   cd productLising
   ```

3. **Install dependencies**:
   ```bash
   # Using bun (recommended for this project)
   bun install
   # OR using npm
   npm install
   ```

4. **Run the development server**:
   ```bash
   bun run dev
   # OR
   npm run dev
   ```

5. **Build for production**:
   ```bash
   bun run build
   # OR
   npm run build
   ```
